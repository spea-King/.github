## ✍️ 프로젝트 소개

### 프로젝트 배경

답변 정답 암기가 아닌 능동적 말하기 연습

요즘 면접은 대면과 부분 비대면을 혼합한 형태로 진행되는 경우가 많습니다. 지원자는 질문을 처음 듣는 순간 당황하거나 머리가 하얘지는 경험을 자주 하며, 시간을 조절하지 못해 답변이 장황해지거나 핵심 없이 끝나는 경우가 많습니다.
사실 답변 내용을 준비하는 것은 어렵지 않지만, 실전에서 이를 구조적으로 말하고 감각적으로 전달하는 능력은 별개의 문제입니다.
또한 면접자는 자신의 이력서, 지원 기업의 공고 및 회사 정보를 기반으로 답변을 적절히 맞추지 못하는 경우가 많습니다.
따라서, 오늘날 지원자에게는 질문을 듣고도 당황하지 않고 능동적으로 답할 수 있는 경험과 훈련이 필수적입니다.

## 📌 주요 기능

### 맥락 기반 면접 질문 생성
- 사용자 이력과 지원 기업·직무 정보를 결합해 실제 면접에서 나올 법한 질문을 생성합니다.
### AI 음성 면접관
- 질문을 음성으로만 제공하여, 대면 면접과 같은 청취·반응 중심 훈련 환경을 구현합니다.
### 답변 페이스메이커 UI
- 제한 시간과 답변 구조(도입–핵심–결론)를 시각적으로 안내해 말의 흐름과 속도를 조절할 수 있도록 돕습니다.
### 답변 분석 및 훈련 피드백
- 답변 내용, 시간 사용, 말하기 속도 등의 데이터를 분석해 점수가 아닌 개선 중심 피드백을 제공합니다.
